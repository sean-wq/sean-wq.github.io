Homebrew基本用法

brew list —列出已安装的软件

brew update —更新Homebrew

brew home *—用浏览器打开

brew info *—显示软件内容信息

brew deps * — 显示包依赖

brew server * —启动web服务器，可以通过浏览器访问http://localhost:4567/ 来同网页来管理包

brew –help —查看brew的帮助

brew install git —安装软件

brew uninstall —卸载软件

brew search —搜索软件

brew upgrade —更新某具体软件

brew cleanup —删除程序，和upgrade一样，单个软件删除和所有程序老版删除。

brew outdated —查看那些已安装的程序需要更新
